# 兵工荣耀 - 微信小程序

## 项目简介

"兵工荣耀"是一个以国防教育为主题的微信小程序，通过"答题闯关-智能题库-社区协作"三级功能体系，构建游戏化、数据化、社交化的国防教育数字化闭环。

## 功能模块

### 1. 答题闯关模块
- **单兵训练**：基础兵工知识入门，5道题目，每题20分
- **战役指挥**：战术策略深度学习，8道题目，每题25分
- **战略推演**：综合能力终极挑战，10道题目，每题30分

### 2. 智能题库模块
- **1000道兵工知识题目**：涵盖军事理论、武器装备、军事历史、战略战术、国防教育五大分类
- 按分类筛选（军事理论、武器装备、军事历史、战略战术、国防教育）
- 按难度筛选（简单、中等、困难）
- 题目搜索功能
- 随机练习功能
- 题目详情与解析

### 3. 社区协作模块
- **动态分享**：发布学习心得和成就
- **组队对抗**：创建战队、加入战队、角色分工
- **沙盘观战**：实时观看战队对战，弹幕互动
- **笔记分享**：学习笔记创建与分享

### 4. 用户中心
- 个人信息展示（头像、昵称、军衔、积分）
- 学习统计（答题数、正确率、学习天数）
- 勋章系统（12种勋章）
- 排行榜（省级、全国、战队）

### 5. 权限管理
- **普通用户**：基础功能使用
- **VIP用户**：优先获取资讯、专属活动
- **管理员**：用户管理、内容审核、功能配置
- **数据管理员**：数据库维护、数据分析、数据安全

## 技术架构

### 前端
- 微信小程序原生开发
- WXML/WXSS/JavaScript
- 微信云开发

### 后端
- 微信云开发
- 云数据库
- 云函数

### 数据库集合
- `users`：用户信息
- `questions`：题目数据
- `posts`：社区动态
- `teams`：战队信息
- `notes`：学习笔记
- `battles`：对战记录
- `records`：答题记录
- `rankings`：排行榜

## 云函数列表

| 云函数 | 功能 |
|--------|------|
| getUserData | 获取用户数据 |
| updateUserPoints | 更新用户积分 |
| addUserMedal | 添加用户勋章 |
| getRankings | 获取排行榜 |
| getQuestions | 获取题目列表 |
| saveRecord | 保存答题记录 |
| createPost | 创建动态 |
| createTeam | 创建战队 |

## 部署步骤

### 1. 环境准备
- 安装微信开发者工具
- 注册微信小程序账号
- 开通云开发服务

### 2. 项目导入
1. 打开微信开发者工具
2. 导入项目目录
3. 填写AppID
4. 配置云环境ID

### 3. 云开发配置
1. 在`app.js`中配置云环境ID
2. 创建云数据库集合
3. 上传并部署云函数

### 4. 数据库初始化
根据`database/schema.json`创建数据库集合和索引

### 5. 云函数部署
1. 右键点击云函数目录
2. 选择"上传并部署：云端安装依赖"
3. 等待部署完成

## 使用说明

### 用户使用流程
1. 首次使用需要登录授权
2. 通过答题闯关获得积分和勋章
3. 在智能题库中练习巩固
4. 在社区分享学习心得
5. 组建战队参与对抗

1. 在用户管理中管理用户
2. 在内容审核中审核动态和笔记
3. 在数据管理中查看统计数据
